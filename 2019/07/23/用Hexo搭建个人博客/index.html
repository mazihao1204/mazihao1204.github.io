本地调试

  1. 安装node.js
    • 查看nodejs和npm版本
  2. 安装cnpm
    • npm install -g cnpm –registry=https://registry.npm.taobao.org
  3. 安装hexo
    • cnpm install -g hexo-cli
    • 验证hexo（hexo -v）
  4. 在目录下建立博客文件夹，同时cd进目录
  5. 安装git
  6. 创建架构
    • hexo init
  7. 开始本地调试
    • hexo s
  8. 创建博文
    • hexo new “我的第一篇博文”
    • 文章会自动生成在/source/_post目录下
  9. 清理并生成
    • hexo clean
    • hexo g
